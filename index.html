Algunos lenguajes de la web

HTML + CSS + Javascript

  • HTML 5

HTML5(HyperText Markup Language , versión 5) es la quinta revisión importante del lenguaje básico de la Word Wild Web, HTML.

Usa la extensión .html

Hoja de estilo en cascada o CSS (siglas en inglés de cascading style sheets) es un lenguaje usado para definir y crear la presentación de un documento estrocturado escrito en HTML.

Usa la extensión css

  • JS

JavaScript(abreviado comúnmente "JS") es un lenguaje de programación interpretado utilizado principalmente en el lad del cliente ( client-side ), implementando como parte de una navegador web permitiendo mejora en la interfaz de usuario y páginas web dinámicas.

Usa la extensión .js
